# My Static Website

A simple and responsive static website built using HTML & CSS. It includes multiple pages:
- Home
- About
- Contact

You can run this project locally using VS Code's Live Server:
[http://localhost:5500/simple/index.html](http://localhost:5500/simple/index.html)

Or run this command in your terminal:
```bash
python -m http.server 5500
```
Then open the link above.

Or simply open the `index.html` file in your browser.
