# To-Do List App

A functional To-Do List application built with HTML, CSS, and JavaScript.

## Features
- Add new tasks
- View task count
- Clear completed tasks
- Interactive UI

## Local Host

You can run this project locally using VS Code's Live Server:
[http://localhost:5500/todo-app/index.html](http://localhost:5500/todo-app/index.html)

Or simply open the `index.html` file in your browser.
